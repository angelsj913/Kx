## 요금제

### ZEFF AI 무료 플랜은 어떤 조건이야?
무료(Free) 플랜은 ₩0입니다. 월 5회 문서 생성, PPT·엑셀 주 1회 체험, 표준 모델 사용, 동시 세션 1개, 서재 저장 5개까지 제공됩니다.

### ZEFF AI Pro 플랜 가격과 조건 설명해줘.
Pro 플랜은 월 ₩9,900입니다. 무료 대비 약 30배 이용량, 우선 처리 큐, 월 문서 생성 150건, 동시 세션 5개, PPT·엑셀 월 20회, 서재 저장 100개, 시험지 분석·유사문제 생성 월 5회, 시험지 제작 월 10회를 제공합니다.

### ZEFF AI Professional 플랜 가격과 조건 설명해줘.
Professional 플랜은 월 ₩19,900입니다. 무료 대비 약 100배 이용량, 다중 AI 라우트·정밀 검수 우선 적용, 월 문서 생성 500건, 동시 세션 20개, PPT·엑셀 월 100회, 서재 저장 500개, 시험지 분석·유사문제 생성 월 25회, 시험지 제작 월 25회를 제공합니다. 팀 계정 연동은 준비 중입니다.

### ZEFF AI 요금제 세 가지를 한눈에 비교해줘.
무료(₩0)는 체험용, Pro(₩9,900/월)는 실무·학습을 본격적으로 쓰는 플랜, Professional(₩19,900/월)은 다중 AI 라우트 정밀 검수와 시험 기능 확장 한도가 필요한 프리미엄 플랜입니다. 위로 갈수록 월 문서 생성 한도, 동시 세션 수, 서재 저장 용량이 커집니다.

## 수학 그래프 기능

### ZEFF AI에서 수학 그래프를 그릴 수 있어?
네. 2D 함수 그래프(y=f(x)), 3D 곡면(z=f(x,y)), 3D 입체 도형(정육면체·각뿔·원뿔·각기둥·원기둥·구·도넛)을 채팅에서 바로 그려줍니다.

### ZEFF AI 3D 도형 그리기는 어떻게 정확도를 보장해?
정육면체·각뿔·원뿔·각기둥·원기둥·구·도넛 같은 표준 도형은 AI가 좌표를 직접 계산하지 않고, 몇 개 숫자 파라미터(반지름·높이·변 개수 등)만 받아 서버에서 정확한 좌표를 계산합니다. 표준 목록에 없는 특수한 모양만 AI가 직접 계산합니다.

### ZEFF AI 2D 그래프는 확대·축소가 되나?
아니요. 2D 그래프는 AI가 이차함수 등을 그리면 항상 그래프 전체 모양이 한 화면에 들어오도록 자동으로 범위를 맞추고, 확대·축소·편집은 지원하지 않는 정적 뷰입니다. 3D는 마우스 드래그로 회전만 가능합니다.

### ZEFF AI 답변에 수식(LaTeX)이 나와?
네. KaTeX 기반으로 분수·제곱근·지수 같은 수식을 실제 수식 표기로 렌더링합니다.

## 채팅 UI

### ZEFF AI 채팅 입력창 위치가 대화 상태에 따라 바뀌어?
네. 대화가 비어 있을 때는 입력창이 화면 위쪽에 있다가, 첫 메시지를 보내면 입력창이 아래로 내려가고 그 위로 대화 내용이 쌓이는 방식으로 자연스럽게 전환됩니다.
