# 수학영역 모의고사 구조·예시 참고자료 (문제 생성용)

이 문서는 시험지 제작(exam-maker)·유사문제 생성(similar-problems) 도구가 **실제 공식 모의고사와 비슷한 구성**의
새 문제를 만들 때 참고하는 자료다. 사용자가 제공한 5개의 공식 모의고사(시도교육청 전국연합학력평가,
한국교육과정평가원 대학수학능력시험 모의평가)에서 직접 확인한 내용이며, 아래 두 가지만 담는다.

1. **구조**: 문항 수·배점·문항 유형 구성 — 실제 5개 시험 전체를 확인해 얻은 정확한 패턴.
2. **예시 문제 일부**: 각 시험에서 뽑은 대표 문제 몇 개(전체 문항이 아님) — 실제 문제의 스타일·난이도·표현
   방식을 보여주는 참고용이다. **정답은 포함하지 않는다**(원본에 답안지가 없어 확인 못함 — 지어내지 않음).

이 자료의 목적은 "이 문제를 그대로 다시 내라"가 아니라, **이런 구성·문체·난이도로 새로운 문제를 만들라**는
참고 틀을 주는 것이다.

## 표준 구조 (2022 개정 교육과정 기준, 5개 시험 공통)

- 총 문항: **공통과목 22문항 + 선택과목 8문항 = 30문항**
- 공통과목(수학Ⅰ, 수학Ⅱ) — 1~22번
  - 5지선다형 15문항(1~15번), 단답형 7문항(16~22번)
  - 배점: 2점·3점·4점 혼합, 15번은 항상 4점 고난도
- 선택과목(확률과 통계 / 미적분 / 기하 중 택1) — 23~30번
  - 5지선다형 6문항(23~28번), 단답형 2문항(29~30번)
  - 27~28번, 29~30번에 고난도(3~4점) 문항 배치
- 시험지 안에서 선택과목별로 섹션이 분리되어 있고(예: "수학 영역(확률과 통계)"), 응시자는 본인이 선택한
  과목 파트만 풂

## 예시 문제 — 2024학년도 5월 전국연합학력평가

### 공통과목
- (1번, 2점) $4^{1-\sqrt{3}} \times 2^{1+2\sqrt{3}}$의 값은?
- (2번, 2점) $\lim_{x\to\infty}(\sqrt{x^2+4x}-x)$의 값은?
- (5번, 3점) $\sin(-\theta)+\cos\left(\frac{\pi}{2}+\theta\right)=\frac{8}{5}$이고 $\cos\theta<0$일 때, $\tan\theta$의 값은?
- (16번, 3점, 단답형) 방정식 $\log_2(x-3)=1-\log_2(x-4)$를 만족시키는 실수 $x$의 값을 구하시오.
- (17번, 3점, 단답형) 함수 $f(x)=(x-1)(x^3+x^2+5)$에 대하여 $f'(1)$의 값을 구하시오.

### 선택과목 예시
- 확률과 통계 (24번, 3점) 다항식 $(ax^2+1)^6$의 전개식에서 $x^4$의 계수가 30일 때, 양수 $a$의 값은?
- 미적분 (23번, 2점) 함수 $f(x)=\sin 2x$에 대하여 $f''\left(\frac{\pi}{4}\right)$의 값은?
- 기하 (23번, 2점) 쌍곡선 $\frac{x^2}{a^2}-\frac{y^2}{36}=1$의 한 점근선이 $y=2x$일 때, 양수 $a$의 값은?

- (1번, 2점) $2^{\frac{1}{3}}\times\sqrt[3]{32}$의 값은?

## 예시 문제 — 2027학년도 대학수학능력시험 6월 모의평가

- (1번, 2점) $\sqrt[3]{9}\times 3^{-\frac{5}{3}}$의 값은?
- (3번, 3점) 두 수열 $\{a_n\},\{b_n\}$에 대하여 $\sum_{k=1}^{5}(2a_k+b_k)=19$, $\sum_{k=1}^{5}(a_k+b_k)=10$일 때, $\sum_{k=1}^{5}a_k$의 값은?
- (18번, 3점, 단답형) 등차수열 $\{a_n\}$에 대하여 $a_6=5$, $a_5=a_2-6$일 때, $a_1$의 값을 구하시오.
- (20번, 4점, 단답형) 1보다 큰 실수 $b$에 대하여 두 함수 $f(x)=b^x$, $g(x)=-\log_b x$의 그래프가 제1사분면에서 만나는 점 P의 좌표를 $(\alpha,\beta)$라 하자. (빈칸 채우기 방식의 조건부 유도 과정 — 대표적인 고난도 논리 전개형 문항)
- 확률과 통계 (28번, 4점) 카드 6장을 뒤집는 반복 시행 확률 문제 (최고난도 문항 — 조건부 확률·반복시행 결합)

## 참고 시 유의사항

- 위 예시는 각 시험의 **일부 문항**만 옮긴 것이며, 실제 전체 문항(30문항)을 재현한 것이 아니다.
- **정답은 포함하지 않았다** — 원본에 답안지가 없어 확인할 수 없었고, 확인 안 된 정답을 지어내 넣지 않았다.
- 새 문제를 생성할 때는 이 예시들을 "그대로 베끼는" 것이 아니라, 배점·난이도·문장 구조·소재(단원)의
  **패턴을 참고해 새로운 조건과 수치로 다시 구성**하는 데 써야 한다.
